# FT-Agent

FT-Agent has two goals:

1. Provide a lightweight node-flow agent architecture for building custom agents.
2. Provide a Fischer-Tropsch catalyst research pipeline built on that architecture.

## Read This README In

- [English](README.en.md)
- [中文](README.zh-CN.md)

## Repository Routes

- Agent runtime: `src/ft_agent/core/`
- LLM integration: `src/ft_agent/llm/`
- Tool system: `src/ft_agent/tools/`
- Fischer-Tropsch pipeline: `src/ft_agent/pipeline/`
- Web UI: `src/ft_agent/web/`
- Examples: `examples/`
